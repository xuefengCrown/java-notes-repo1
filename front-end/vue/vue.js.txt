
##vue特性
1. 声明式渲染

2. 条件与循环
v-if v-for

3. 双向数据绑定
<input type="text" v-model="model-name" />

4. 处理用户输入
为了与用户和你的应用进行交互，我们可以用 v-on 指令添加一个事件监听器，通过它调用在 Vue 实例中定义的方法。

5. 组件化应用构建
可以定义一个个组件，在 vue 页面中引用组件，这个功能非常适合构建大型应用。


##MVVM
M: Model
V: View
VM: View Model(Dom Listeners, Directives)
Dom 操作封装在指令中
应用逻辑全部是数据操作

VM 要做的工作就是 Vue.js 要承担的。
